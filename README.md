# Pulse Metronome Privacy Policy

Pulse Metronome does not collect, store, transmit, or share personal information from users.

## Data Handling
- No personal data is collected.
- No personal data is shared with third parties.
- App settings such as tempo, presets, and sound preferences are stored only on the user's device.
- Pulse Metronome does not use a remote server to upload user data.
- Pulse Metronome does not use analytics, advertising SDKs, or tracking technologies.

## Permissions
Pulse Metronome may request notification permission in order to show playback status and controls while the metronome is running.
This permission is used only for core app functionality and not for advertising or tracking.

On some Android devices, the app may also guide the user to battery optimization or autostart settings so playback notifications can work reliably.
This does not involve the collection, transmission, or sharing of personal information.

## Children
Pulse Metronome does not knowingly collect personal information from children.
